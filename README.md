A simple python project to pictorially represent an atom (Bohr's Model) upon the input of its atomic number.
It also displays other information like the mass number and the number of neutrons by fetching the data from a local CSV file.
For the graphics, I have used the turtle library in python, which required me to come up with an algorithm to comply with Aufbau's and Bohr-Bury rule.
It was a lot of fun! ;)
